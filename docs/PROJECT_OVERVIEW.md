# Project Overview — Henty Lodge Command Centre

## Summary

The **Henty Lodge Command Centre** is a dashboard for running a boutique accommodation business. It is not intended to replace Cloudbeds or accounting software. It sits above those tools and turns operational, financial and marketing data into a simple daily view.

The first build should be a practical MVP with manual/sample data, clean UI and AI-generated suggestions.

---

## Target user

The primary user is the Henty Lodge owner/operator.

They need to quickly know:

- Who is arriving today
- Who is leaving today
- Which rooms/spaces are occupied
- What tasks need doing
- Which guests need follow-up
- Whether revenue and occupancy are on track
- Which dates need marketing attention
- What campaign/content action to take next

---

## Core problem

Accommodation businesses often have information scattered across:

- PMS/booking systems
- Email/messages
- Spreadsheets
- Google Analytics
- Google Business Profile
- Google Ads
- Social media
- Notes and task lists

The dashboard brings this together into one simple operating view.

---

## Product promise

> Henty Lodge Command Centre helps the owner run the lodge each day, spot empty dates early, follow up guests properly, and create marketing campaigns faster.

---

## MVP product statement

A web dashboard that displays today’s bookings, guests, tasks, reviews, revenue, marketing metrics and AI suggestions using manual/sample data, with the foundation for later Cloudbeds and Google integrations.

---

## Key outcomes

The MVP should help the owner:

1. Save time each morning
2. Avoid missed check-ins, reviews and cleaning tasks
3. See booking gaps before they become a problem
4. Draft campaigns for empty dates
5. Understand direct vs platform bookings
6. Track reviews and guest follow-ups
7. Make marketing decisions from one place

---

## Success metrics

Useful product metrics later:

- Time saved each week
- Occupancy increase
- Direct booking percentage
- Review request completion rate
- Campaigns created from booking gaps
- Revenue generated from AI-suggested campaigns
- Repeat guest bookings
- Website-to-booking conversion rate

---

## MVP persona

**Owner/operator:**

- Wants a simple overview, not complex analytics
- Needs fast decisions
- Does not want to manually check five different tools each morning
- Wants AI help but still wants control
- Values direct bookings, reviews, and repeat guests
